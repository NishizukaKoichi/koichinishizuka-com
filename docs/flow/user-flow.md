# User Flow Definition

## entry_channels
Epoch の基本原則と禁止事項を読む説明ページへの到達。
（登録前は説明ページのみ閲覧可能）

## first_success
初回 record がサーバ時刻で確定し、record_hash が生成され不可逆保存された時。
（decision_made / decision_not_made の確定）

## payment_consideration
他人の Epoch を判断材料として読む必要が生じた瞬間。
（Time Window / Read Session の選択が必要になる）

## post_purchase
他人の Epoch を読み切った後に再訪し、自己の記録が継続していることを確認するために戻る。
