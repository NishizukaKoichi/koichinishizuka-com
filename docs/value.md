# Value Definition

## who
判断や行為の履歴が評価・印象・物語に歪められる環境に身を置いてきた人。
採用、協業、信頼形成などで説明や自己演出に疲弊している人。
過去を良くも悪くも正当化せずに残したい人。

## situation
過去の判断や行動について説明や評価を求められた瞬間。
沈黙や見送った選択が「なかったこと」にされそうになった瞬間。
相手の言葉や実績ではなく、時間の積み重なりそのものを判断材料として見たい／見せたい瞬間。

## pain
履歴が後から編集・要約・物語化されてしまう。
語らなかったこと、選ばなかったことが記録に残らない。
評価やスコアに変換された瞬間に文脈と時間が失われる。
「自分の判断を、他人の言葉で説明される」ことへの不信。

## after_paying
他人の Epoch を判断材料として読む瞬間に、編集不能・評価なし・沈黙欠損なし・時間順序絶対であることが確定する。
情報量ではなく、不確実性の除去に対価を払っていると明確に理解できる。
